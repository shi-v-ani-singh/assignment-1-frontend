# Assignment-1 Frontend

This is the **frontend** of Assignment-1 built with **Next.js + TypeScript**.

## Features

- Login and Registration pages
- Welcome page after successful login
- Clean interactive UI using **Framer Motion**
- Simple navigation between pages
- Form validation and loading states

## Pages

- `/register` → Registration page
- `/login` → Login page
- `/welcome` → Welcome page after login

## How to Run

1. Clone the repo:

```bash
git clone https://github.com/shi-v-ani-singh/assignment-1-frontend



2. Install dependencies:

npm install
# or
yarn


3. Start the development server:

npm run dev
# or
yarn dev


4. Open http://localhost:3000
in your browser.

Note: Frontend communicates with the backend at http://localhost:3001.
